# PHYSIQUEIQ Coaching Rules

## Purpose

This document defines the decision-making principles used by PHYSIQUEIQ.

The platform does not generate random workouts.

Every recommendation must be the result of structured coaching rules supported by scientific evidence.

---

# Rule Structure

Every rule contains:

- Trigger Conditions
- Decision
- Evidence
- Confidence Score
- Exceptions
- Priority

---

# User Inputs

The Decision Engine evaluates:

- Primary Goal
- Secondary Goal
- Training Experience
- Sport Experience
- Competition Status
- Equipment
- Gym Access
- Recovery
- Sleep
- Activity Level
- Lifestyle
- Injuries
- Available Training Days
- Age
- Sex
- Body Weight
- Height
- Performance History

# Example Rule

IF

Goal = Fat Loss

AND

Recovery = Poor

THEN

Reduce Weekly Volume

Increase Recovery

Maintain Protein Intake

Lower Cardio Intensity

---

# Design Principle

Every coaching recommendation must be explainable.

The system should always be able to answer:

Why was this recommendation made?

Which evidence supports it?

What user information influenced it?

Which rule generated it?
