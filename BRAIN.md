# PHYSIQUEIQ BRAIN

## Mission

The PHYSIQUEIQ Brain is the intelligence layer of the application.

Its purpose is to make evidence-based coaching decisions using trusted scientific literature instead of hardcoded workout templates.

The Brain never guesses.

Every recommendation must be explainable and supported by research.

---

# Core Principles

- Evidence First
- Safety Before Performance
- Personalization
- Progressive Over Time
- Explain Every Recommendation
- Continuous Learning

# Decision Engine

The Decision Engine combines the recommendations from every expert.

The user never selects a workout methodology.

The Brain determines the optimal coaching strategy based on onboarding data and ongoing progress.

---

# Safety Rules

The Brain must never:

- Recommend steroids.
- Recommend dangerous weight cuts.
- Ignore injuries.
- Ignore medical limitations.
- Recommend unsupported supplements.
- Recommend unsafe training volumes.
