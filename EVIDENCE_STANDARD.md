# PHYSIQUEIQ Evidence Standard

## Mission

Every recommendation produced by PHYSIQUEIQ must be supported by the highest quality evidence available.

The system should always prioritize stronger evidence over weaker evidence.

# Decision Rules

The AI should always use the highest available evidence.

When multiple studies disagree:

- Prefer Meta Analyses.

If no Meta Analysis exists:

- Prefer RCTs.

If no RCT exists:

- Use Position Stands.

If no Position Stand exists:

- Use Expert Consensus.

Never recommend information that is unsupported unless clearly labelled as emerging evidence.
